# Task 07 - Criar estrutura de dominio catalog

## Objetivo
Criar o container de dominio `libs/catalog`.

## Comandos
```bash
mkdir -p libs/catalog
```

## Descricao detalhada
- Cria a raiz do dominio de catalogo/produtos.

## Resultado esperado
- Pasta `libs/catalog/` existente.

## Criterio de aceite
- `libs/catalog` visivel no workspace.

## Erros comuns e correcao
- Em PowerShell, usar `New-Item -ItemType Directory -Path libs/catalog -Force`.

## Checkpoint
Se aceito, ir para `task-08-create-orders-data-access.md`.
