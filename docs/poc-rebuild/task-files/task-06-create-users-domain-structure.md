# Task 06 - Criar estrutura de dominio users

## Objetivo
Criar o container de dominio `libs/users`.

## Comandos
```bash
mkdir -p libs/users
```

## Descricao detalhada
- Cria a raiz de dominio para autenticacao/perfil.

## Resultado esperado
- Pasta `libs/users/` existente.

## Criterio de aceite
- `libs/users` visivel no workspace.

## Erros comuns e correcao
- Em PowerShell, usar `New-Item -ItemType Directory -Path libs/users -Force`.

## Checkpoint
Se aceito, ir para `task-07-create-catalog-domain-structure.md`.
